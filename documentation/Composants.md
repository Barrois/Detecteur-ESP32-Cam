# Detecteur-ESP32-Cam

## Composants

- un ESP32-CAM

- un micro type Max981

- (une carte Sd)

## Montage

La sortie du Max981 est reliée sur l'entrée GIPO0 de l'ESP32-CAM.

Un interrupteur permet de déconnecter les deux au démarrage de l'ESP-32 afin qu'il ne passe pas en mode "flash".
